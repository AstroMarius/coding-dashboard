# Daily

Cartella per registrare attività giornaliere, note e risultati.

Struttura:
- `daily/YYYY-MM-DD/` — cartelle per giorno
- Dentro ogni cartella giornaliera: `today.md` (nota giornata), eventuali artefatti o file di log

Uso rapido:
1. Crea una nuova cartella con la data odierna `YYYY-MM-DD`.
2. Aggiungi `today.md` con le attività svolte.
3. Commit e push per tenere traccia.
